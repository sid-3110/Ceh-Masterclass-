# 🔐 CEH MasterClass

> A free, open-source, beginner-friendly learning platform covering all 20 EC-Council CEH v12 modules + a SIEM bonus module.



## 🌐 Live Demo
👉 [sid-3110.github.io/ceh-masterclass](https://sid-3110.github.io/ceh-masterclass)

## 📚 What's Inside

| Feature | Details |
|---|---|
| 📦 Modules | 20 CEH v12 + 1 SIEM Bonus |
| 📊 Diagrams | Visual SVG diagrams per module |
| 🧪 Labs | Step-by-step Kali Linux lab instructions |
| ✅ Progress | Tracks completed modules (localStorage) |
| 🔍 Search | Filter by category or keyword |
| 📱 Responsive | Works on mobile, tablet, and desktop |

## 🚀 How to Use

**Option 1 — Live (no setup):**  
Visit the GitHub Pages link above.

**Option 2 — Local:**  
```bash
git clone https://github.com/sid-3110/ceh-masterclass.git
cd ceh-masterclass
# Open ceh.html in any browser — no server needed
```

## 🛠️ Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Fraunces, Outfit, JetBrains Mono)
- SVG diagrams (no external charting library)
- localStorage for progress persistence
